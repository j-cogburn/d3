# UX & UI
**Consulted at:** SHAPE · PROVE

- **Nielsen's 10 heuristics** — visibility of status, match to real world, user control, consistency, error prevention, recognition over recall, flexibility, minimalist design, help users recover, help/docs.
- **Laws of UX** — Fitts (target size/distance), Hick (choice cost), Gestalt (grouping), Jakob (match conventions).
- **Atomic design + design tokens** — compose from tokens/components so quality is the default.
- **Mobile-first / responsive** — design the constrained case first.
- **First-class states** — empty, loading, error, offline, zero-result, long-content are designed, not afterthoughts.

**Gate:** UI directives carry the state-coverage done-when row.
