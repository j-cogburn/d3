# Principle library

Proven, elite-level standards by domain. D3 injects the relevant file(s) into an
agent's brief at the phase where they bite (see the "Consulted at" line in each).
Every consequential decision must **cite the principle it rests on** — reviews
check conformance to a known-good pattern, not taste.

Defaults are tuned for **SaaS products**. A project may override any file by
editing its copy in `.d3/principles/`.
