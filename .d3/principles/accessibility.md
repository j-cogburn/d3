# Accessibility
**Consulted at:** SHAPE done-when · PROVE

- **WCAG 2.2 AA** (ISO/IEC 40500:2025) — target standard; 2.1 AA is the legal floor (US ADA Title II, Apr 24 2026).
- **Perceivable / Operable / Understandable / Robust** — the four principles.
- **Keyboard navigation** — every interactive element reachable and operable without a mouse.
- **ARIA + semantics** — correct roles/labels; prefer native elements.
- **Contrast** — >= 4.5:1 text, 3:1 large text / UI components.

**Gate:** PROVE runs an a11y pass on changed surfaces (pass/fail per surface).
